What is the binary system?

The binary or base-2 system is a positional numbering system in which all quantities are represented based on two numbers, 0 and 1. In computing, a binary digit (0 or 1) is called a bit, and a grouping of 8 bits corresponds to one byte.

This system is the basis of Boolean Algebra (by George Boole â€” English mathematician), which allows you to do logical and arithmetic operations using only two digits or two states. All digital electronics and information technology are based on the binary system and Boolean logic, which allows the representation of numbers and characters by digital electronic circuits and the performance of logical and arithmetic operations. Thus, this system is used as a natural numbering system for digital computers, which work internally with two voltage levels.

For more information access: Wikipedia - Binary numbering system

What is the decimal system?

The decimal system is a positional numbering system that uses base ten.

The fundamental principle of the decimal system is that ten units of any order form one of the next higher order. After the orders, the constituent units of the numbers are grouped into classes, where each class has three orders, where each order has a special name, being identical to the same orders of the other classes.

The first class, that of units, has the orders of hundreds, tens and units. The order of the units corresponds to the numbers from one to nine. The order of tens corresponds to the numbers ten (one ten) to ninety (nine tens), each of these numbers being ten times the corresponding number in the previous order. Hundreds correspond to numbers from one to nine hundreds. Analogously, each of these numbers corresponds to ten times the corresponding one in the previous order.

For more information access: Wikipedia - Decimal
